# Cryptocurrency
Cryptocurrency Price Trend and Market Behavior Analysis

📌 Project Overview

This project focuses on analyzing cryptocurrency market data to understand price trends, market behavior, and trading activity. The cryptocurrency market is highly volatile, making it essential to extract meaningful insights from historical data to support better decision-making.

🎯 Objectives
Analyze cryptocurrency price trends over time
Understand market behavior such as volatility and trading activity
Perform data cleaning and preprocessing
Conduct exploratory data analysis (EDA)
Build meaningful visualizations
Derive insights and recommendations from the data

📂 Dataset Information
Source: CoinGecko API
Time Period: January 2025 – March 2026
Data Type: Time-series / Market snapshot data
Size: ~500+ rows, 12 columns

🔑 Key Features
Coin Name & Symbol
Current Price
Market Capitalization
Total Volume
Price Change (24h)
High & Low (24h)
Category
Volatility
Trend
Timestamp

🧹 Data Preprocessing
Handled missing values using median and mode imputation
Removed duplicate records
Renamed and standardized column names
Converted data types (Timestamp to datetime)
Extracted new features (Year, Month, Day, Hour)
Applied log transformation to reduce skewness

📊 Exploratory Data Analysis (EDA)
Summary statistics and distribution analysis
Skewness and kurtosis evaluation
Correlation analysis between key variables
Identification of trends, patterns, and anomalies

📈 Visualizations
Created using Python (Matplotlib, Seaborn) and Power BI:
Bar Chart → Top cryptocurrencies by market capitalization
Pie Chart → Category-wise market distribution
Scatter Plot → Relationship between price and trading volume
Trend Analysis → Monthly price and volume patterns
Heatmap → Feature correlation analysis

📊 Dashboard (Power BI)
Interactive dashboard with filters (Coin, Category, Date)
Visual comparison of market trends and activity
User-friendly layout for better decision-making

🔍 Key Insights
Market is highly volatile with frequent price fluctuations
A few cryptocurrencies dominate based on market capitalization
Higher trading volume often indicates stronger market presence
Category-wise distribution shows concentration in specific sectors
Strong relationships exist between price, volume, and volatility

💡 Recommendations
Use market capitalization instead of price to identify dominant coins
Monitor trading volume for better investment insights
Focus on stable categories to reduce risk
Use trend analysis for short-term decision-making

🚀 Future Enhancements
Use larger and more recent datasets
Implement predictive models for price forecasting
Build advanced interactive dashboards
Automate data pipeline and reporting

🛠️ Tools & Technologies
Python (Pandas, NumPy)
Data Visualization (Matplotlib, Seaborn)
Power BI
Jupyter Notebook

📌 Conclusion
This project transforms raw cryptocurrency data into meaningful insights by analyzing trends, patterns, and relationships. It helps in understanding market behavior and supports data-driven decision-making.
